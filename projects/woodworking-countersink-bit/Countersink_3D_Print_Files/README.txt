BESPOKE PRECISION CLICK-LOCK COUNTERSINK
3D Printable Prototype Files
================================================

FILES INCLUDED:
- countersink_prototype.scad  (OpenSCAD source - editable)
- body.stl                    (Main body - 303 SS in production)
- collar.stl                  (Depth adjustment collar)
- ratchet_ring.stl            (60-tooth ratchet ring)
- cam_lever.stl               (Cam lock lever)
- foot_carrier.stl            (Bearing carrier)
- uhmw_foot.stl               (Non-marring foot ring)
- cutter_head.stl             (Cutter head body)
- collet.stl                  (Split collet for drill bit)
- collet_nut.stl              (Knurled collet nut)

PRINT SETTINGS:
- Layer height: 0.12mm for fine detail, 0.20mm for fit-check
- Material: PETG recommended (PLA for quick prototyping)
- Infill: 50% for structural parts, 100% for small parts
- Supports: Yes, for cutter_head and body
- Scale: 1:1 (actual size, all dimensions in mm)

NOTES:
- Threads are simplified for visual reference
- For functional prototype threads, add 0.2mm tolerance
- The body.stl may need mesh repair (non-manifold warning)
- Open the .scad file in OpenSCAD to modify dimensions
- Change part="exploded" to see exploded view
